# NasiyaSavdo

DUR tokenini nasiya asosida sotish va sotib olish platformasi — Optimism Mainnet.

## Ishga tushirish

```bash
npm install --legacy-peer-deps
npm start
```

Brauzerda `http://localhost:3000` da ochiladi. MetaMask kengaytmasi kerak.

## Build (saytga yuklash uchun)

```bash
npm run build
```

`build/` papkasi hosil bo'ladi — uni Vercel, Netlify yoki o'z serveringizga yuklang.
